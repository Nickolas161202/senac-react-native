inicializa um repositório local
git init

Cria uma branch local
git branch -M main

conceta um repositório local a um remoto
git remote add origin https://github.com/instrutorvilson/senac-react-native.git

adiciona arquivos ao commit
git add >arquivos<

Efetua o commit no repositorio local
>git commit -m "Primeiro commit"

Tranfere os arquivos comitados localmente para o remoto
git push -u origin main


verifica qual repositorio remoto está conectado
git remote -v
